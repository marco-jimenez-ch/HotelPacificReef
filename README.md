## Descripción del Proyecto

Sistema web de gestión de reservas hoteleras desarrollado para el 
Hotel Pacific Reef. Permite a los turistas consultar disponibilidad, 
reservar habitaciones y realizar pagos de garantía en línea, mientras 
entrega al equipo administrativo herramientas de gestión, reportes y 
control de ocupación.
